# MyLove

This is a project for my love. Demo: [https://deer.dedog.top](https://deer.dedog.top/).

## Prerequisite

- Python 3.10

```bash
$ pip install Flask
```

## Run

- Clone this repository
- Start Python HTTP server

```bash
$ cd MyLove/
$ python app.py
```

Now, your entire website is running on http://0.0.0.0:3414.
